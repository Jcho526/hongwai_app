## 🌟 核心功能

### API 端点一览

```
┌─────────────────────────────────────────────────┐
│ GET /health                                     │
│ 🔍 健康检查                                    │
│ 返回：{"status":"healthy"}                     │
│ 用途：验证服务是否运行                         │
└─────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────┐
│ GET /api/v1/status                             │
│ 📊 服务状态查询                                │
│ 返回：版本、支持格式、文件大小限制              │
│ 用途：获取 API 配置信息                        │
└─────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────┐
│ POST /api/v1/generate-pdf ⭐ 核心功能          │
│ 📄 单个 PDF 生成                               │
│ 输入：热成像图片 + 患者信息                     │
│ 输出：PDF 报告文件                             │
│ 耗时：~2.8 秒                                  │
└─────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────┐
│ POST /api/v1/batch-generate ⭐ 批量处理       │
│ 📦 批量 PDF 生成                               │
│ 输入：多个患者数据列表                         │
│ 输出：JSON 包含每个患者的结果                   │
│ 耗时：~2.8s × 患者数                          │
└─────────────────────────────────────────────────┘
```

## 💼 集成方式

### 方案 A：Python 客户端（最简单）
```python
from thermal_client import ThermalAPIClient

client = ThermalAPIClient('http://localhost:5000')
result = client.generate_pdf('thermal.png', 'patient_name', ...)

if result['success']:
    print(f"✅ PDF: {result['pdf_path']}")
```

### 方案 B：JavaScript/Vue.js
```javascript
const formData = new FormData();
formData.append('file', imageFile);

const response = await fetch('http://localhost:5000/api/v1/generate-pdf', {
    method: 'POST',
    body: formData
});

const pdf = await response.blob();
// 处理或下载 PDF
```

### 方案 C：cURL 命令行
```bash
curl -X POST http://localhost:5000/api/v1/generate-pdf \
  -F "file=@thermal.png" \
  -F "patient_name=赵女士" \
  --output report.pdf
```

## 🆘 常见问题解决

```
问题 #1：无法启动服务
  └→ 检查 Python 版本 (3.7+)
  └→ 检查依赖安装 (pip install -r requirements.txt)
  └→ 直接运行查看错误信息

问题 #2：PDF 转换失败
  └→ 安装 LibreOffice (https://www.libreoffice.org)
  └→ Windows: 使用官方安装程序
  └→ 重启 PowerShell 窗口

问题 #3：端口被占用
  └→ 使用命令查看：netstat -ano | findstr :5000
  └→ 杀死占用进程：taskkill /PID <PID> /F
  └→ 或改用其他端口

快速诊断：
  └→ 运行：python test_api.py
  └→ 自动测试所有端点
  └→ 详细的彩色输出
```
