# 红外热图分析项目文档

## 1. 项目概述

本项目是一个基于红外热图的健康分析系统，结合中医理论和现代图像处理技术，用于分析人体红外热图，识别异常区域，并提供健康风险评估。

### 1.1 核心功能
- 红外热图预处理与特征提取
- 热图对称性分析
- 颜色分布分析与温度映射
- 异常区域检测
- 基于中医理论的健康风险评估
- PDF文档内容提取（辅助功能）

### 1.2 技术栈
- **编程语言**：Python 3.x
- **核心库**：OpenCV (cv2)、NumPy、PIL (Pillow)
- **辅助库**：PyPDF2（用于PDF内容提取）
- **数据存储**：JSON（热图字典和知识库）

## 2. 项目结构

## 3. 核心模块

### 3.1 ThermalImageAnalyzer 类

`ThermalImageAnalyzer` 是项目的核心类，负责热图的分析和处理。

#### 主要功能：
- **初始化**：加载热图字典和知识库
- **图像预处理**：调整尺寸、转换颜色空间
- **特征提取**：对称性分析、颜色分布、异常区域检测、温度统计
- **颜色温度映射**：将热图颜色转换为温度类型（高温、正常、低温）

#### 关键方法：
- `preprocess_image()`：热图预处理
- `extract_features()`：提取热图特征
- `_calculate_symmetry()`：计算左右对称性
- `_analyze_color_distribution()`：分析颜色分布
- `_detect_abnormal_regions()`：检测异常区域
- `_calculate_temperature_stats()`：计算温度统计信息

### 3.2 热图字典 (thermal_dict.json)

热图字典是项目的核心知识库，包含以下内容：

- **knowledge_base**：基础理论、分析方法、风险提示
- **thermal_dictionary**：常见热图模式和对应的健康状况
- **body_regions**：人体各个部位的定义

### 3.3 PDF内容提取工具 (read_pdf.py)

用于从PDF文件中提取文本内容，生成对应的txt文件，为知识库提供补充资料。

## 4. 工作流程

1. **初始化分析器**：加载热图字典和知识库
2. **图像预处理**：读取并预处理红外热图
3. **特征提取**：分析热图的对称性、颜色分布等特征
4. **异常检测**：基于颜色分布检测高温或低温异常区域
5. **健康评估**：根据特征和知识库评估健康风险

## 5. 核心算法

### 5.1 对称性分析
- 计算热图左右两侧的差异
- 生成对称性得分（0-1，1表示完全对称）
- 用于识别不对称的异常区域

### 5.2 颜色分布分析
- 将热图像素分类为不同颜色（白、红、紫、黄、绿、蓝、黑）
- 计算各颜色的占比
- 基于颜色分布判断温度类型

### 5.3 异常区域检测
- 基于颜色分布检测高温或低温异常
- 当高温区域占比超过30%时，视为异常
- 当低温区域占比超过50%时，视为异常

## 6. 中医理论应用

项目结合中医理论，将热图特征与健康状况对应：

- **颜色意义**：白色（炎症）、红色（气血）、紫色（气血）、黄色（代谢较缓）、绿色（寒症）、蓝色（寒湿）
- **温度意义**：高温（血多，瘀血，静脉回流不畅）、低温（血少，局部缺血，动脉供血不足）
- **异常区域**：手、脚、肚子等部位的异常温度

## 7. 应用场景

1. **中医诊断辅助**：帮助中医医生分析患者的热图，辅助诊断
2. **健康风险评估**：通过热图分析评估潜在健康风险
3. **体质分析**：基于热图特征分析个人体质类型（如上热下寒）
4. **疾病预防**：早期发现潜在健康问题，实现治未病

## 9. 使用方法

### 9.1 热图分析

```python
from thermal_analyzer import ThermalImageAnalyzer

# 初始化分析器
analyzer = ThermalImageAnalyzer()

# 预处理热图
image = analyzer.preprocess_image('path/to/thermal_image.jpg')

# 提取特征
features = analyzer.extract_features(image)

# 分析结果
print('对称性得分:', features['symmetry_score'])
print('颜色分布:', features['color_distribution'])
print('异常区域:', features['abnormal_regions'])
print('温度统计:', features['temperature_statistics'])
```

```python
# 并生成对应的_content.txt文件
```
