# 热图分析Agent设计方案

## 1. 需求分析
设计一个能够根据热图字典和看图知识对人体热图进行分析的Agent，输出基于热图字典的分析结果。

## 2. 核心功能
- 热图输入处理：支持常见图像格式（jpg、jpeg、png、bmp）的人体热图输入
- 热图特征提取：分析热图的对称性、异常点和颜色分布
- 热图字典匹配：根据提取的特征与热图字典进行匹配
- 分析结果输出：生成结构化的热图分析报告

## 3. 技术架构

### 3.1 模块设计
- **输入模块**：处理热图文件输入，支持单张或多张热图
- **预处理模块**：热图图像预处理，包括尺寸调整、颜色校准等
- **特征提取模块**：
  - 对称性分析：检测左右身体的温度分布差异
  - 异常点检测：识别不该出现高温/低温的区域
  - 颜色分析：提取各区域的颜色特征，对应温度含义
  - 区域定位：确定异常区域的身体部位
- **字典匹配模块**：将提取的特征与热图字典进行匹配，查找相似的典型图例
- **分析报告模块**：生成结构化的分析结果，包括异常区域、可能的健康风险、建议等

### 3.2 实现技术
- **编程语言**：Python
- **图像处理**：OpenCV、PIL
- **特征提取**：基于像素颜色分析和区域分割
- **字典管理**：将热图字典转换为结构化数据，便于匹配

## 4. 实现步骤

### 4.1 数据准备
1. 将热图字典转换为结构化数据，包括：
   - 身体部位分类（头、颈、胸、腹、背、四肢等）
   - 典型热图特征（颜色分布、温度异常模式）
   - 对应健康风险说明

2. 建立热图分析知识库，整合看图知识中的理论：
   - 温度含义映射
   - 颜色与健康状况对应关系
   - 常见风险提示

### 4.2 核心功能实现
1. **热图预处理**：
   - 统一热图尺寸和分辨率
   - 提取温度数据（如果热图包含温度信息）
   - 颜色空间转换，便于分析

2. **特征提取**：
   - 对称性分析：计算左右身体对应区域的温度差异
   - 异常点检测：设定正常温度范围，识别超出范围的区域
   - 区域分割：将热图划分为不同身体部位
   - 颜色分布统计：计算各区域的颜色占比

3. **字典匹配算法**：
   - 基于区域颜色特征的相似度计算
   - 身体部位匹配
   - 异常模式匹配

4. **分析报告生成**：
   - 异常区域定位和描述
   - 对应热图字典匹配结果
   - 健康风险提示
   - 基于看图知识的建议
